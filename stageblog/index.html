Week 3 Stage - VWE Automotive

Deze week ben ik begonnen met het maken van een moeilijkere template op maandag, dinsdag ochtend heb ik die afgekregen en heb ik een nieuwe template gekregen om door te oefenen. Die heb ik dezelfde dag laat op de dag nog afgekregen. Daarna met ik begonnen met een grotere template aan het einde van de dag. Op woensdag heb ik de hele dag gewerkt aan een grote template waar er heel veel dingen zijn die je moet veranderen en daar ook er goed op letten dat het er precies hetzelfde uitziet als het voorbeeld.

Week 2 Stage - VWE Automotive

In de 2e week ben ik begonnen aan het maken van een klantenwebsite template, dit houd in dat ik deze keer voor een echt bedrijf een site template maak, deze wordt nog niet naar het bedrijf toegestuurd, maar het is meer een soort van simulatie van hoe het zou zijn als ik echt voor een klant een site moet maken. Op woensdag ben ik begonnen met het afwerken van de klantenwebsite en daarna moest ik gaan oefenen met het maken van een carousel header, daar heb ik vervolgens de rest van de dag op getraind. Op donderdag heb ik vervolgens een opdracht gekregen om een template te maken en erbij gebruiken wat ik heb geleerd bij het maken van de carousel header. Die moet dan vervolgens worden nagekeken en krijg ik feedback en kan ik de fouten er uit halen en opnieuw inleveren.

Posted by Almar Gemmel on March 4, 2019

Week 1 Stage - VWE Automotive

In de eerste week ben ik op maandag en een deel van dinsdag begonnen aan het maken van de bootstrapkaart zodat ik met alle componenten kon oefenen. Op woensdag en donderdag ben ik vervolgens begonnen met het maken van een website template zodat ik leer hoe ik vanaf een basis template een website moet maken voor een autobedrijf. Vervolgens moest ik de site responsive maken zodat de site er op telefoons en tablets ook goed uit ziet.
